# Speller

A bilingual South African classroom-friendly word game based on Spellie.

Speller is a modified non-commercial fork of the original Spellie project by Christopher Porter and David Hirtle.

This fork adapts the game for:

- English and Afrikaans play
- Grade-based gameplay for Grade 2, Grade 3, and Grade 4
- South African word lists and British/South African English spellings
- Reward emojis and classroom-friendly visuals
